Step 2 of 4

Anthropic API Key

This key lets your agents call Claude AI. You get $5 free credits — enough for the entire project.

1. Go to console.anthropic.com

2. Sign up or log in

3. Click API Keys → Create Key

4. Copy the key (starts with sk-ant-)

5. Open your .env file and add:

ANTHROPIC_API_KEY=sk-ant-your-key-here

Step 3 of 4

GitHub Token

This token lets your agents read PRs, CI runs, and deployment history from GitHub.

1. Go to GitHub → Settings → Developer settings

2. Click Personal access tokens → Tokens (classic)

3. Generate new token — set expiry to 90 days

4. Check these 3 scopes only:

public_repo   repo:status   repo_deployment

5. Copy the token (starts with ghp_) and add to .env:

GITHUB_TOKEN=ghp_your-token-here
